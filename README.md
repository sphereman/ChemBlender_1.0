ChemBlender is a Blender-based plugin for professional visualization of molecules and other micro-nano structures. It supports Blender versions 4.1â€“4.4. Before using the plugin, install the third-party Python library RDKit. You can install it by entering the following command in Blender's Python Console:

import sys

import subprocess

python_exe = sys.executable

subprocess.call([python_exe, '-m', 'pip', 'install', 'rdkit'])
